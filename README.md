# Copernicus Marine

EU Copernicus Marine Environment Monitoring Service providing free and open access to global ocean data including temperature, salinity, currents, sea level, sea ice, and biogeochemical variables via the Copernicus Marine Toolbox API, OGC WMTS tile service, and CSW catalogue API.

- **Website:** https://marine.copernicus.eu/
- **Data Store:** https://data.marine.copernicus.eu/products
- **Help Center:** https://help.marine.copernicus.eu/
- **Toolbox Docs:** https://toolbox-docs.marine.copernicus.eu/
- **GitHub:** https://github.com/mercator-ocean/copernicus-marine-toolbox
- **Registration:** https://data.marine.copernicus.eu/register

## APIs

| API | Base URL | Auth |
|-----|----------|------|
| Copernicus Marine Toolbox API | https://data.marine.copernicus.eu | Username/password (free account) |
| Copernicus Marine WMTS API | https://wmts.marine.copernicus.eu/teroWmts | None |
| Copernicus Marine CSW Catalogue API | https://csw.marine.copernicus.eu/geonetwork/csw-MYOCEAN-CORE-PRODUCTS/eng/csw | None |
